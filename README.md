# AI Resume Screening System

## Internship
Sqrock IT Solutions

## Project

This project compares resumes with a job description using TF-IDF and Cosine Similarity.

## Tools Used

- Python
- Scikit-learn
- TF-IDF
- Cosine Similarity

## Features

- Reads Job Description
- Reads Multiple Resumes
- Calculates Resume Match Score
- Ranks Candidates

## Output

Resume 1 : 67.56%

Resume 3 : 7.54%

Resume 2 : 44.51%

## Conclusion

The project automatically identifies the best matching resume for the job description.
